# U.S. Airline Delay and Performance 2015 Analysis 

## Project Overview
This project analyzes U.S. airline delays, cancellations, airline performance, airport performance, and temporal flight trends using PostgreSQL and Power BI.

## Business Objective
- Understand causes of delays and cancellations.
- Compare airline and airport performance.
- Analyze trends by month, day, and time of day.
- Provide data-driven recommendations to improve efficiency and passenger experience.

## Tools Used
- PostgreSQL
- pgAdmin
- Power BI
- SQL
- DAX

## Datasets Used
- Flights Dataset
- Airlines Dataset
- Airports Dataset

## Key KPIs
- Total Flights
- On-Time Performance Rate
- Average Departure Delay
- Average Arrival Delay
- Cancellation Rate
- Delay Type Contribution

## Dashboards
### 1. Overview Dashboard
Shows total flights, OTP rate, average delays, cancellation rate, delay type impact, and monthly OTP trend.

### 2. Airline Performance Dashboard
Compares airlines by flight volume, OTP rate, cancellation rate, and average arrival delay.

### 3. Airport Performance & Temporal Trends
Analyzes airport traffic, best/worst months, best/worst days, peak delay time, and busiest flight time slots.

## Key Insights
- Late Aircraft Delay is the largest delay contributor.
- Hawaiian Airlines has the best OTP performance.
- Spirit Airlines shows poor OTP performance.
- October has the best OTP.
- February has the worst OTP.
- June is the peak delay month.
- Morning is the busiest flight time slot.
- Evening has the highest average arrival delay.

## Recommendations
- Improve aircraft turnaround management.
- Strengthen evening operations to reduce delay propagation.
- Prepare contingency plans for high-risk months such as February and June.

## Conclusion
This project provides interactive dashboards and insights to support data-driven decision-making for airlines and airport authorities.
